Projects:

  • Modular Multiplicative Inverse Calculator A script to calculate and show the equations used to find the GCD then multiplicative inverse (if there is one) for a number in a modular ring up to 50,000,000. Finding the modular multiplicative inverse is used in practical applications to speed up the encryption process of RSA. Thanks to Adam McCarthy (@aem1269) for helping test the script.
  • MaLV (Machine Language Visualizer) Updating a work in progress JavaScript Deterministic Finite Automaton simulator to make the layout and functionality more user friendly along with adding Nondeterministic functionality. Developed for the Honors Into to Computer Science Theory course with team members Caleb Kofahl (@caleb42david) and Aaron Stadler (@aaron13michael).
  • Toolshare A website written in Django, CSS and HTML simulating a community where people can rent and loan tools. Functionality includes user creation, profiles, and messaging, tool creation, editing, loaning and renting, along with administrative functionality of creating user holding "sharezones", banning users and deleting "sharezones", users, or tools. Developed for the Intro to Software Engineering class with team members Corban Mailloux, Adam McCarthy (@aem1269), Michael Washburn, and Bryon Wilkins.
